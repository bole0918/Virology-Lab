# images/

這個資料夾存放網站使用的圖片（人物大頭照、活動照片等）。

## 上傳新圖片

1. 點右上角「**Add file → Upload files**」
2. 拖曳圖片到頁面
3. 最下方按「**Commit changes**」

## 在試算表中引用

在 Google Sheets 對應欄位填入路徑：

- `members` 分頁 `avatar` 欄 → 例如 `images/profile.jpg`
- `highlights` 分頁 `image` 欄 → 例如 `images/lab-dinner-2026.jpg`

## 圖片建議

- **格式**：JPG / WebP / PNG
- **尺寸**：寬度 400–1200px
- **大小**：<500KB（可用 https://tinypng.com 壓縮）
- **命名**：英文 + 連字號，例如 `pi-photo.jpg`、`student-wang.jpg`
