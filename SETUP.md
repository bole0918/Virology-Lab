# 王文宏實驗室網頁 — 內容更新指南

這份文件說明如何用 **Google Sheets**（不寫程式）更新網頁上的所有內容。

---

## 🧭 架構總覽

```
Google Sheets（你維護內容）
        ↓
  公開 CSV 連結
        ↓
  index.html 載入時自動 fetch
        ↓
   網頁顯示最新內容

GitHub repo /images/  ←  存放圖片（網頁拖曳上傳）
```

**只有兩個地方要更新：**
1. **文字資料** → Google Sheets
2. **圖片** → GitHub repo 的 `images/` 資料夾

---

## 📋 階段一：建立 Google Sheets 試算表（一次性設定，約 15 分鐘）

### 步驟 1：建立新試算表

1. 用 Chrome 打開 [https://sheets.new](https://sheets.new)，會自動建立新試算表
2. 把名稱改成「**分子病毒學實驗室資料庫**」（左上角）

### 步驟 2：匯入 6 個分頁

repo 裡的 `data/` 資料夾有 6 個 CSV 檔，每一個對應一個分頁：

| CSV 檔               | 分頁名稱要改成 | 內容           |
| -------------------- | -------------- | -------------- |
| `news.csv`           | `news`         | 最新消息       |
| `publications.csv`   | `publications` | 著作目錄       |
| `highlights.csv`     | `highlights`   | 獲獎與活動     |
| `history.csv`        | `history`      | 歷史專區       |
| `members.csv`        | `members`      | 實驗室成員     |
| `projects.csv`       | `projects`     | 計畫 / 專利    |

**匯入步驟（每個 CSV 重複一次）：**

1. 試算表選單列：「**檔案** → **匯入**」
2. 點「**上傳**」分頁，把 `data/news.csv` 拖進去
3. 「匯入位置」選 **「插入新工作表」**
4. 「分隔符類型」選 **「逗號」**
5. 按「**匯入資料**」
6. 匯入完成後，雙擊下方分頁標籤把名稱改成 `news`（**全部小寫，跟表格中第一欄一樣**）

對其他 5 個 CSV 重複以上步驟。完成後試算表會有 6 個分頁，外加一個原本的 `工作表1`（可以右鍵刪除）。

### 步驟 3：發佈到網路

1. 選單列：「**檔案** → **共用** → **發佈到網路**」
2. 在彈出的對話框中：
   - 左下選「**整份文件**」
   - 右邊選「**逗點分隔值 (.csv)**」
   - 勾選「**有變動時自動重新發佈**」
3. 按「**發佈**」→ 確認「**確定**」

### 步驟 4：複製試算表 ID

看你的試算表網址，會像這樣：

```
https://docs.google.com/spreadsheets/d/1AbC2dEfGhIjKlMnOpQrStUvWxYz0123456789/edit
                                       └────────── 這一段就是 ID ──────────┘
```

把網址中間那一段（`/d/` 跟 `/edit` 中間的那串字）複製起來，傳給我。

## 🖼️ 階段二：圖片管理

### 上傳新圖片

1. 用 Chrome 打開 [https://github.com/bole0918/Virology-Lab](https://github.com/bole0918/Virology-Lab)
2. 點進 `images/` 資料夾
3. 右上角「**Add file** → **Upload files**」
4. 把照片拖進去，最下方按「**Commit changes**」

### 在試算表中引用圖片

在 `highlights` 分頁的 `image` 欄位填上路徑，例如：

```
images/lab-dinner-2026.jpg
```

**圖片建議：**
- 格式：JPG / WebP / PNG
- 大小：寬度建議 1200px 內，檔案 <500KB（網頁載入快）
- 命名：用英文 + 連字號（不要用中文或空格），例如 `lab-dinner-2026.jpg`

## ✏️ 日常更新流程

### 📢 news（最新消息）

1. 打開 Google Sheets → 切到 `news` 分頁
2. 在最上面插入一列（最新的放最前面）
3. 填欄位：

| 欄位       | 範例            | 說明                          |
| ---------- | --------------- | ----------------------------- |
| date       | 2026 · January  | 顯示在卡片上的日期            |
| type       | pub             | 看下方對照表                  |
| title      | 新論文發表...   | 標題                          |
| body       | 內容說明...     | 描述                          |
| permanent  | FALSE           | 是否常駐顯示（招募類設 TRUE） |

**`type` 對照表：**

| 試算表填入 | 顯示徽章 | 圖示 | 適用情境               |
| ---------- | -------- | ---- | ---------------------- |
| `pub`      | 論文發表 | 📄   | 新論文 accepted、刊登  |
| `event`    | 公告     | 🤝   | 一般公告、產學合作邀請 |
| `recruit`  | 長期招募 | 🎓   | 招生、徵助理           |

> 💡 招募類消息建議把 `permanent` 設成 `TRUE`，這樣它會被釘選顯示為紫色卡片。

---

### 📄 publications（著作目錄）

| 欄位         | 值                                       |
| ------------ | ---------------------------------------- |
| year         | 2026                                     |
| first_author | TRUE / FALSE（是否第一作者，會顯示徽章） |
| article_type | 看下方對照表（留空=不顯示）              |
| icon         | 📄 / 🦠 / 🧬 等任意 emoji                  |
| title        | 論文標題                                 |
| authors      | 作者列表                                 |
| journal      | 期刊資訊                                 |
| metrics      | 影響因子等（可空白）                     |

**`article_type` 對照表：**

| 試算表填入 | 顯示文字         | 顏色  | 適用情境                             |
| ---------- | ---------------- | ----- | ------------------------------------ |
| `original` | Original Article | 🔵 藍 | 原創研究、實驗、回溯分析、方法開發   |
| `review`   | Review           | 🟣 紫 | 綜述、Systematic Review、回顧性文章  |
| `book`     | Book Chapter     | 🟢 綠 | 專書 / 書章                          |
| `letter`   | Letter           | ⚫ 灰 | Letter / Editorial                   |
| `case`     | Case Report      | 🟠 橘 | Case Report                          |
| （留空）   | 不顯示徽章       | -     | 不確定或不想顯示時                   |

---

### 🌟 highlights（獲獎與活動 — 顯示在輪播卡片）

| 欄位  | 值                                |
| ----- | --------------------------------- |
| type  | 看下方對照表                      |
| year  | 2026                              |
| emoji | 🏆 / 📸 / ✈️ 等任意 emoji          |
| image | `images/檔名.jpg`（沒有的話留空） |
| title | 標題                              |
| desc  | 描述                              |

**`type` 對照表（highlights 與 history 共用）：**

| 試算表填入   | 顯示徽章   | 預設圖示 | 適用情境                       |
| ------------ | ---------- | -------- | ------------------------------ |
| `award`      | 🏆 獲獎    | 🏆       | 各類獎項、補助、優秀獎         |
| `activity`   | 📸 活動    | 📸       | 實驗室聚餐、新生訓練、年度活動 |
| `conference` | ✈️ 研討會  | ✈️       | 國內外會議發表、出國研討會     |

> 💡 `emoji` 欄可以填特殊符號（例如 🥈 銀牌、🌏 國際會議），蓋過預設圖示。

---

### 📖 history（歷史專區 — 顯示在「歷史專區」分頁的年份分組）

| 欄位  | 值                                       |
| ----- | ---------------------------------------- |
| year  | 2021                                     |
| type  | `award` / `activity` / `conference`      |
| emoji | 🏆 / 📸 / ✈️ 等                            |
| title | 標題                                     |
| desc  | 描述                                     |

> 💡 `type` 對照表跟 highlights 完全一樣，看上面那張。
> 💡 「比較舊」的年份事件適合放這裡，「最新」的活動放 highlights。

### 👥 members（實驗室成員）

| 欄位    | 值                                              |
| ------- | ----------------------------------------------- |
| group   | 看下方對照表（決定顯示在哪個分區）              |
| avatar  | emoji（如 🧬）**或**圖片路徑（如 `images/profile.jpg`）|
| name    | 中文姓名                                        |
| name_en | 英文姓名 / 學位                                 |
| role    | 職稱                                            |
| topic   | 研究題目（學生用，PI 與 alumni 留空）           |
| since   | 加入年份                                        |
| desc    | 詳細介紹（**只有 PI 用**，學生留空）            |
| dest    | 畢業去向（**只有 alumni 用**，其他人留空）      |

**`group` 對照表：**

| 試算表填入 | 顯示分區          | 適用對象          |
| ---------- | ----------------- | ----------------- |
| `pi`       | 上方大卡片        | 主持人（PI）      |
| `phd`      | 🎓 博士研究生     | 博士生            |
| `ms`       | 📚 碩士研究生     | 碩士生            |
| `ug`       | 🌱 大學部專題生   | 大學部專題學生    |
| `ra`       | 🔭 研究助理       | 研究助理          |
| `alumni`   | 🎓 畢業校友       | 已畢業成員        |

**`avatar` 用法：**

- 填 emoji：`🧬`、`👨‍🔬`、`🌱` 等 → 顯示成 emoji
- 填圖片路徑：`images/profile.jpg` → 顯示成照片（自動裁切成圓形）

### 🏛 projects（研究計畫 / 專利）

| 欄位     | 值                                                    |
| -------- | ----------------------------------------------------- |
| category | 看下方對照表（決定顯示在哪個分頁）                    |
| icon     | emoji 圖示                                            |
| title    | 計畫 / 專利名稱                                       |
| period   | 起迄期間（例如 `2024 – 迄今`，**只有 current/past 用**）|
| source   | 經費來源（**只有 current/past 用**）                  |
| desc     | 描述                                                  |
| tags     | 標籤，用 `\|` 分隔（**只有 current/past 用**），例如 `國科會計畫\|基礎研究` |
| number   | 專利字號（**只有 patents 用**）                       |
| region   | 專利地區（**只有 patents 用**）                       |
| year     | 專利年份（**只有 patents 用**）                       |
| licensee | 授權對象（**只有 patents 用**，可空白）               |

**`category` 對照表：**

| 試算表填入 | 顯示分頁     | 適用情境                       |
| ---------- | ------------ | ------------------------------ |
| `current`  | 🟢 現行計畫  | 進行中的研究計畫               |
| `patents`  | 📜 專利授權  | 已獲准的發明專利、授權合約     |
| `past`     | 🕰 過往計畫  | 已完成的研究計畫               |

> ⚠️ **`tags` 欄非常重要**：必須用直立線 `|`（而不是逗號）分隔多個標籤，不然會被誤認為 CSV 欄位分隔符。

---

## ⚠️ 注意事項

1. **不要刪掉第一列的欄位名稱**（`date, type, title...`），程式靠它們對應欄位。
2. **不要重新命名分頁**（要保持 `news` / `publications` 等小寫名稱）。
3. **CSV 不接受逗號開頭** — 如果某欄要留空，留白即可。
4. **`tags` 欄用 `|` 分隔**（不是逗號），例如 `國科會計畫|基礎研究|臨床轉譯`。
5. 改完到網頁更新，**最多需要 5 分鐘**（Google 發佈延遲），可以強制重新整理（Ctrl+Shift+R）試試。

## ❓ 常見問題

**Q：改了試算表但網頁沒更新？**
A：Google 公開 CSV 有快取，等 5 分鐘再強制重新整理。如果還沒更新，去「檔案 → 共用 → 發佈到網路」確認還在發佈狀態。

**Q：不小心改壞了某一列？**
A：Google Sheets 有完整修訂紀錄，「檔案 → 版本記錄 → 查看版本記錄」可以還原。

**Q：要新增一整個新區塊（不在這 6 個分頁裡）？**
A：傳訊息給 Claude 讓他幫你加。
